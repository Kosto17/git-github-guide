# Git, GitHub & Version Control

A beginner-friendly repository that explains **Git**, **GitHub**, **Version Control Systems (VCS)**, **SVN**, **Centralized vs Distributed Version Control**, and the **core Git workflow**. This repository is designed as a learning resource with simple explanations, examples, and commonly used Git commands.

---

## 📖 About This Repository

This repository was created to learn and understand the fundamentals of Git and GitHub. It covers the concepts of version control, how Git works, why GitHub is used, and the complete workflow of managing a project using Git.

Whether you're a student, a beginner in programming, or someone starting with open-source development, this repository provides an easy-to-follow introduction to the most important Git concepts.

---

## 🎯 Objectives

By studying this repository, you will learn:

- What Version Control is
- Why Version Control is important
- Types of Version Control Systems
- Centralized vs Distributed Version Control
- What SVN is
- What Git is and why it is widely used
- What GitHub is and how it works
- The Core Git Workflow
- Common Git commands
- Understanding commits and commit messages
- Best practices while using Git and GitHub

## 🔄 Core Git Workflow

```text
Create Project
      │
      ▼
git init
      │
      ▼
git status
      │
      ▼
git add .
      │
      ▼
git commit -m "Initial Commit"
      │
      ▼
Create GitHub Repository
      │
      ▼
git remote add origin <repository-url>
      │
      ▼
git branch -M main
      │
      ▼
git push -u origin main
```
